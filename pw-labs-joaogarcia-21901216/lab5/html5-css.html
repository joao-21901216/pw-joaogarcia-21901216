HTML 5

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

CSS

Tipo de Seletor Usado? Comentário e Uso
universal check Altera a estilização de todos os elementos HTML a que a stylesheet é aplicada.
Foi usado para alterar a font base de todas as páginas para um custom.
tipo check Altera a estilização apenas de elementos HTML do tipo especificado.
Foi usado para estilizar vários elementos de formas diferentes.
classe check Altera a estilização apenas de elementos HTML da classe especificada.
Foi usado para definir fonts e criar as classes usadas para colocar imagens sobrepostas.
id check Altera a estilização apenas de elementos HTML com o id especificado.
Foi usado para alterar a fonte de um dos elementos do menu animado.
atributo check Altera a estilização apenas de elementos HTML do tipo especificado que façam uso do atributo especificado.
Foi usado para estilizar links, borders e texto.
pseudo-classe check Altera a estilização apenas de elementos HTML que estejam num estado especificado.
Foi usado para alterar a estilização dos vários estados dos links e algum texto.
pseudo-elemento check Altera a estilização apenas de elementos HTML na parte do mesma especificada.
Foi usado para colocar icons antes e depois dos links no menu de navegação.

Combinadores de Seletores Usado Comentário e Uso
Descendentes
(Ex: div p) 		check Altera a estilização de todos os elementos p dentro de um elemento div, por exemplo.
Foi usado para estilizar várias elementos de formas diferentes.
Filhos
(Ex: div > p) 		check Altera a estilização de todos os elementos p que estejam diretamente dentro de div, por exemplo, no entanto se estiverem dentro de outro elemento dentro de div não são afetados.
Foi usado para estilizar links em figcaptions, fieldsets em forms e h4 dentro de sections.
Adjacentes
(Ex: div ~ p) 		check Altera a estiliziação de todos os elementos p colocados após um elemento div, por exemplo. Foi usado para estilizar figcaptions após imgs.
Imediatamente Adjacentes
(Ex: div + p) 		check Altera a estilização apenas do primeiro elemento p que se encontre imediatamente após um elemento div, mas não se houver qualquer elementos entre eles.
Foi usado para estilizar elementos details após elementos a e elementos p após elementos sup.
